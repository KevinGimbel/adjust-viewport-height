Use the power of the viewport height unit everywhere

Supported Browsers

14+ 4+ 4+ 10.6+ 8+*
All mentioned browsers tested with BrowserStack on OS X and Windows.
iOS seems not to work (according to my BrowserStack tests), Android does work pretty well.

Download

Minified (1.1kB) Developer (5.1kB)

Usage

Include the adjustviewportheight.js, call the function with a data-attribute of your choice (I prefere data-vunit="adjust").

Then set the data-vunit="adjust" attribute to the elements you want to adjust.

Modern browsers, not IE 8-10

                
            # Works in all browsers (expect IE 8-10, but IE 11), sets the default height: 100vh;
            <div data-vunit="adjust"><div>

All browsers, custom height

Optional: if IE 8-10 mustn't be supported there is a data-vheight attribute for custom heights. For IE 8-10 Support read on.

                
            # Works in all browsers, equals height: 15vh; in 
            <div data-vunit="adjust" data-vheight="15"><div>

IE 8-10

Unfortunately in IE 8-10 you need to specify a data-vheight attribute, otherwise the height will not be set to 100vh (which is default)

IE 8-10 fix

                
    # Works in all browsers and IE 8-10, sets height: 100vh;
    <div data-vunit="adjust" data-vheight="100"><div>
