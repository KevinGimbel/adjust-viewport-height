Use the power of the viewport height (vh) everywhere

Supported Browsers

14+ 6+ 5.1+ 11.1+ 8+

Usage

Include the adjustviewportheight.js in your page, call the function with a data-attribute of your choice (I prefere data-vunit="adjust").

Then set the data-vunit="adjust" attribute to the elements you want to adjust.

Modern browsers, not IE 8-10

                
            # Works in all browsers (expect IE 8-10, but IE 11), sets the default height: 100vh;
            <section data-vunit="adjust"><section>

All browsers, custom height

Optional (if IE 8-10 mustn't be supported) there is a data-vheight attribute for custom heights.

                
            # Works in all browsers, equals height: 15vh; in 
            <section data-vunit="adjust" data-vheight="15"><section>

Works perfectly with every browser

but not Internet Explorer.

Unfortunately in 8-10 you need to specify a data-vheight attribute, otherwise the height will not be set to 100vh (which is default)

IE 8-10 fix

                
    # Works in all browsers and IE 8-10, sets height: 100vh;
    <section data-vunit="adjust" data-vheight="100"><section>
